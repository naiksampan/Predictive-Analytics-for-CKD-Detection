Predictive-Analytics-for-CKD-Detection

Predictive analytics project for early detection of Chronic Kidney Disease using machine learning, data preprocessing, feature engineering, and performance evaluation for clinical decision support.

create a conda environment: conda create -n CKD_Env python=3.8 pandas numpy scipy matplotlib seaborn plotly scikit-learn

activate the envirnoment: conda activate CKD_Env

Install requirements.txt

Approch is given in brief.pdf

The data handling and EDA is given in CKD_analysis.ipynb
